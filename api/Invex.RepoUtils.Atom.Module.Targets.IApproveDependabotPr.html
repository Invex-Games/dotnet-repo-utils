ApproveDependabotPr

Enables auto-merge on the target Dependabot pull request. The target validates that the pull request was opened by Dependabot before resolving the pull request and issuing the GraphQL mutation to enable auto-merge using the merge strategy.

DependabotEnableAutoMergePat

A GitHub personal access token with the permissions required to enable auto-merge on pull requests. A dedicated PAT is required because the default workflow token cannot trigger the downstream workflows needed to complete an auto-merge.
