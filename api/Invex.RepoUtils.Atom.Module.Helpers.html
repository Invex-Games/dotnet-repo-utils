Namespace Invex.RepoUtils.Atom.Module.Helpers

Interfaces

IApiSurfaceHelper

Provides functionality for analyzing the public API surface of the repository by diffing the tracked API definition files between two commits and classifying the resulting changes as either major (removals) or minor (additions) breaking changes.

ICopilotReviewHelper

Provides functionality for waiting until GitHub Copilot has finished reviewing a pull request. GitHub does not currently expose a native way to block auto-merge on a Copilot review, so this helper polls the pull request's review state until Copilot's review has landed (or a timeout elapses).

IDocFxHelper

Provides functionality for generating, serving, and publishing DocFX documentation for the repository. The helper builds the static documentation site, can host it locally for previewing, and publishes the generated output to a project's gh-pages branch for GitHub Pages hosting.

IGithubPrHelper

Exposes the GitHub pull request context that targets operating on a specific pull request require.

INugetPackageUnlistHelper

Provides functionality for discovering and unlisting prerelease packages that have been superseded by a newly published version. Versions are discovered through the NuGet flat-container API and unlisted via the feed's HTTP DELETE endpoint, with a summary written to the Atom build report.

IPrBreakingChangeHelper

Orchestrates the end-to-end breaking change check for a pull request: it locates the most recent release to use as a baseline, identifies breaking changes against it, validates that the version has been bumped appropriately, and reports the result back to GitHub as a check run.
